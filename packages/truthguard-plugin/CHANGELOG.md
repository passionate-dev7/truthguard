# Changelog

All notable changes to the TruthGuard plugin will be documented in this file.

## [0.0.1] - 2025-11-26

### Added
- Initial release of TruthGuard multi-modal deepfake detection plugin
- 5 MCP tools: deepfake_detect, fact_check, content_monitor, validator_coordinate, creator_protect
- Multi-modal detection system (visual, audio, text)
- Verification swarm with specialized agents
- Community consensus engine
- Evidence collection and blockchain storage
- Database schema with 8 tables
- Python ML scripts for visual, audio, and text detection
- EfficientNet-B7 for visual analysis
- Audio deepfake detection with spectral analysis
- RoBERTa-based text AI detection
- Multi-modal fusion engine with 3 methods (weighted_average, voting, deep_fusion)
- REST API endpoints (/detect, /fact-check, /health)
- Comprehensive test suite
- DKG integration with ClaimReview and CreativeWork schemas
- Creator attribution and protection system
- Real-time content monitoring
- Validator reputation system

### Features
- 85-95% multi-modal detection accuracy
- Support for image, video, audio, text, and mixed content
- Cryptographic evidence chains
- Community validator coordination
- Fact-checking with source validation
- Attribution proof registration
